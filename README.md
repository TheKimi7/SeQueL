# SeQueL

A Magisk module that adds the `sqlite3` binary to your Android system. (>A13)

## How to Install
1. Download the `SeQueL.zip` from the **Releases** section.
2. Open the **Magisk App** and go to **Modules**.
3. Tap **Install from storage** and select the zip.
4. Reboot your phone.

## How to Use
Open a terminal (or ADB shell) and type:

su
sqlite3
